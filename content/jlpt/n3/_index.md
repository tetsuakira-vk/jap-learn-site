## Key N3 Grammar Patterns

| Pattern | Meaning | Example |
|---------|---------|---------|
| 〜ば〜ほど | the more ~ the more ~ | 練習すればするほど上手になる。The more you practice, the better. |
| 〜たら | if / when ~ (then) | 着いたら電話して。Call me when you arrive. |
| 〜ても | even if / even though | 雨が降っても行きます。I'll go even if it rains. |
| 〜のに | even though ~ (frustration) | 頑張ったのに負けた。I worked hard but lost. |
| 〜らしい | apparently / seems like | 彼は来ないらしい。Apparently he's not coming. |
| 〜はずだ | should be / expected to be | 彼女はもう着いているはずだ。She should be here already. |
| 〜ことにする | decide to do ~ | 毎日運動することにした。I decided to exercise daily. |
| 〜ようにする | try to do ~ / make effort to | 早く寝るようにしている。I try to go to bed early. |
| 〜てくる | to start doing / to come back after doing | 日本語が上手になってきた。My Japanese is getting better. |
| 〜ていく | to continue doing / to go and do | これからもっと勉強していく。I'll keep studying from now on. |
| 〜ばかり | just / only / nothing but | 彼はゲームばかりしている。He does nothing but play games. |
| 〜だけでなく | not only ~ but also | 日本語だけでなく文化も学んだ。I learned not only Japanese but culture too. |
